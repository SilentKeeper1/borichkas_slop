# 🍕 Borichkas Slop

Онлайн-сервіс доставки піци, створений трьома учасниками під час навчання у [GoITeens Python Web](https://goiteens.ua/).  
🔗 [Перейти на сайт](https://borichkas-slop.onrender.com/)

---

## 🚀 Функціонал
- Авторизація та особистий кабінет
- Меню з піцою з БД
- Кошик і оформлення замовлення
- Інтерактивна мапа з перевіркою доставки
- Бонусна система
- Адмін-панель

---

## ⚙️ Технології
Flask • SQLAlchemy • SQLite • Jinja2 • HTML/CSS/JS • Leaflet.js
